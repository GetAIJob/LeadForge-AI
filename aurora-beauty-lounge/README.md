# Aurora Beauty Lounge

**Type:** Full-service beauty salon & aesthetics studio  
**Location:** Florence, Italy (demo)  
**Pipeline:** LeadForge AI by GrowthSite Lab

## Design System
- **Heading font:** Cormorant Garamond (400, 600)
- **Body font:** DM Sans (300, 400, 500)
- **Primary bg:** #FAFAF8 (cream)
- **Accent:** #C4A882 (champagne gold)
- **Dark sections:** #1E1B18
- **Mood:** Warm spa luxury, editorial serif headings, champagne gold accents

## Files
- `index.html` — Complete single-page site, 10 sections
- `css/styles.css` — Mobile-first stylesheet, CSS custom properties
- `js/main.js` — Header scroll, burger menu, scroll reveal, form validation

## Images Required (replace placeholders)
- `assets/images/gallery-1.webp` through `gallery-6.webp` — Treatment/results photos
- `assets/images/about-studio.webp` — Studio interior portrait
- All images: WebP format, 800×600 recommended (about-studio: 600×750)
